NAME
       myman


SYNOPSIS
       myman [BUILTIN]


DESCRIPTION
       The  myman  utility  displays  the  manual pages of the builtins of the
       project.

       Those builtins are:
       - myman
       - exit
       - cd
       - echo
       - setenv
       - unsetenv
       - env
       - set
       - unset
       - export
       - local

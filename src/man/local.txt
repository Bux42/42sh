NAME
       local /- print all local variables


SYNOPSIS
       local - write local variables to the standard output


DESCRIPTION
       This utility writes all local variables line by line


OPTIONS
       None.


EXIT STATUS
       The local utility exits 0 on success and >0 if an error occurs.

NAME
       local /- print all variables locales


SYNOPSIS
       local - write locales variables to the standart output


DESCRIPTION
       This utility just writes all locales variables line by line


OPTIONS
       None.


EXIT STATUS
       The local utility exits 0 on success and >0 if error occurs.
